## Description

Parses text in a string as XML document and returns its root node.

The input to this method is expected to be a [Data](/reference/api/Data). If you have a string input, use [XML.parse()](/reference/api/XML/parse) instead.

## Syntax

``` js
XML.decode(data)
```
